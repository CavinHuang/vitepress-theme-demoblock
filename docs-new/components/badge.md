## API

### Badge 属性

| 参数     | 说明                            | 类型           | 默认值  |
| -------- | ------------------------------- | -------------- | ------- |
| value    | 徽标内容                        | string、number | -       |
| max      | 最大值，超过最大值会显示 {max}+ | number         | 0       |
| dot      | 是否展示红点                    | boolean        | `false` |
| show-zero | 标记为 `0、-0` 时是否显示       | boolean        | `true`  |



### Badge 插槽

| name    | 说明       |
| ------- | ---------- |
| default | 自定义内容 |
