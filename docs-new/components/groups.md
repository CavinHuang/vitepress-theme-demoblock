## API

### Groups 属性

| 参数   | 说明                                 | 类型                     | 默认值       |
| ------ | ------------------------------------ | ------------------------ | ------------ |
| data   | 原数据                               | Array、Object            | －           |
| limit  | 可展示几条,传0或不传,默认展示全部 　 | number                   | -            |
| orient | 展示的朝向                           | `horizontal`、`vertical` | `horizontal` |

### Groups 插槽

| 插槽名字 | 说明           |
| -------- | -------------- |
| default  | 自定义按钮内容 |
