## API

### Empty 属性

| 参数  | 说明      | 类型   | 默认值     |
| ----- | --------- | ------ | ---------- |
| text  | 内容      | string | `暂无数据` |
| image | image地址 | string | -          |
| icon  | icon名字  | string | -          |



### Empty 插槽

| name    | 说明       |
| ------- | ---------- |
| default | 自定义内容 |
